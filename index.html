SQLMate

Rapidly generate a DAO for SQLite

View on GitHub Download .zip Download .tar.gz

Quick Mission:

I tossed this together to speed-up the creation of allot of SQLite3 for Java / JDBC recently. These classes will generate the basic C.R.U.D operations for a DAO. Schema creation & drop as well.

Thought I would share the package.

Jump Start:

The demonstration on how to use this package is in main.java. See the source code for more information.

  • Sample output file from same is in SqlTestTable.java.
  • The implementation is designed for super-fast growth.
